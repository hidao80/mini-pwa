## 説明

このプロジェクトは、最小限のプログレッシブウェブアプリ（PWA）のサンプルです。

## 使い方

このプロジェクトをローカルで実行するには、https_server.pyを実行し、ブラウザで<https://localhost:8443>にアクセスしてください。
